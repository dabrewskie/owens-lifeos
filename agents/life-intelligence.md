## Cross-Domain Analysis Framework

```
HEALTH (Base Layer)

    ↓ funds
```

**Key insight:** This is a circular system. Health degradation doesn't just affect the gym — it degrades cognitive performance, which degrades work quality, which slows career advancement, which pressures finances, which creates stress, which degrades family presence, which degrades health. Everything is connected.
